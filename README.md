# Polat İskele Project

A modern and responsive company website built for **Polat İskele**, a scaffolding and construction services business.

This project was developed as a real-world business website prototype to present company services, references, and contact information through a clean and professional user interface.

## 🚀 Live Demo

Live website: https://www.polatiskele.com.tr/

## 📌 Project Overview

Polat İskele Project is a responsive corporate website designed to improve the online presence of a construction/scaffolding company. The main goal of the project is to create a modern, user-friendly, and mobile-compatible website that clearly presents the company’s services and builds trust with potential customers.

The project focuses on frontend development, reusable component structure, responsive design, and clean UI implementation.

## 🛠️ Tech Stack

* React
* TypeScript
* Vite
* Tailwind CSS
* shadcn/ui
* Radix UI
* React Router
* Lucide React

## ✨ Features

* Modern responsive landing page
* Mobile-friendly navigation menu
* Reusable React components
* Service presentation sections
* Company statistics section
* Contact-focused layout
* Clean and professional UI design
* Fast development setup with Vite
* Production-ready build structure

## ⚙️ Getting Started

To run this project locally:

```bash
npm install
npm run dev
```

Then open the local development URL shown in the terminal.

## 🏗️ Build

To create a production build:

```bash
npm run build
```

To preview the production build locally:

```bash
npm run preview
```
